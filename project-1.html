Portfolio Website Prototype

As part of my Web Design and Development course at University, I had a project with a research study. The subject of my study was the creativity and the User Experience of portfolio websites. To showcase my study, I created a high-fidelity prototype of a portfolio website for a fictional student seeking employment.

  • 2019
  • Web Design, UX
  • Webflow, Photoshop, Indesign
see website

The prototype was created with Webflow and the visuals were made with Photoshop and Photomosh. I had to create a poster (made with InDesign ) to present my project to other students. Later, I made an evaluation with few participant to improve the UX of the website.
